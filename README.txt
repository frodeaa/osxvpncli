DESCRIPTION:
  This is a wrapper to help you start and stop your OSX VPN connections from
  the command line.

SETUP:
  First you must create some VPN connections that you wish to manipulate. You
  must do that from "System Preferences -> Network".

USAGE:
  vpn [OPTION] <vpn_name>

OPTIONS:
  status          Show the status of each of your VPN connections.
  connect         Establish a connection to a VPN.
  disconnect      Disconnect an established connection.
